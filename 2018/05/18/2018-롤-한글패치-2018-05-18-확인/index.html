[2018] 롤 8.10 한글패치 2018/05/18 확인

2018-05-18
League of legends, 롤, 리그 오브 레전드, 한글 패치

해외 롤 한글 패치

8.10 패치후 서버 주소가 변경되어 새로운 파일들로 패치를 진행하셔야 합니다.

본인이 플레이 중인 서버용 파일이 존재하지 않는다면 저에게 이메일로 서버명과 system.yaml 파일을 보내주시고 댓글로 원하는 서버 이름을 남겨주세요, 최대한 빠르게 이 페이지에 업데이트 하도록 하겠습니다.

가레나 서버 관련

가레나 서버는 다른 서버와 달리 가레나에서 자체 운영중인 서버라 패치 방법이 달라 한글패치가 현재 불가능 합니다.

패치 적용

포스트 가장 아래에 존재하는 서버별 파일에서 본인의 서버에 해당하는 system_XXX.yaml 파일을 다운로드 합니다.
다운로드후 system_XXX.yaml 파일을 system.yaml 로 이름변경 해주세요.

이후 롤이 설치된 폴더로 이동하는데, 롤을 실행할때 사용하는 아이콘을 우클릭후 -> 속성 -> 대상찾기 를 누르시면 하위폴더로 바로 이동 가능합니다.

롤이 설치된 폴더 이름이 Riot Games\League of Legends\ 라고 했을시,
Riot Games\League of Legends\RADS\projects\league_client\releases 로 들어갑니다.
그리고 releases 폴더 안에 존재하는 폴더들중 가장 숫자가 큰 폴더로 들어가셔서 deploy 폴더로 들어갑니다.

2018-05-18 오세아니아 서버 기준 경로는
Riot Games\League of Legends\RADS\projects\league_client\releases\0.0.0.138\deploy 입니다.

해당 폴더에 다운받고 이름변경한 system.yaml을 넣어줍니다. 기존에 존재하는 파일 위에 덮어 쓰세요.
그리고 system.yaml을 복사해서 "Riot Games\League of Legends\" 폴더에다가도 붙혀넣어 줍니다.

해당 작업이 끝나면 롤을 재시작 해주세요, 그러면 한글 패치가 진행될텐데요, 업데이트 작업이 끝나면 위의 작업을 한번 더 해줍니다.
system.yaml 파일을 옮기고 롤을 처음 실행시 한국 서버로 접속되는것이 정상이며, 패치가 끝나면 다시 system.yaml파일을 덮어씌워주고 재접속해야 원하는 서버로 접속 가능합니다

한글 파일들을 다운로드 받느라 덮어씌워놨던 system.yaml 파일이 바뀌는 경우가 많습니다.

로그인이 안될시 댓글에 문의 주세요.

서버별 파일

서버 추가를 원하시는 분들은 댓글에 원하시는 서버를 남겨주시고, 이메일로 본인이 사용하는 system.yaml 파일과 원하는 서버를 적어서 보내주세요.
augusdn@gmail.com 로 보내주시면 됩니다.

8.12 버전용 한글 패치는 새로 포스팅 올렸습니다.

새로운 포스팅을 봐주세요
