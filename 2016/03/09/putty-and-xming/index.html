PuTTY 란?

PuTTY is an SSH and telnet client, developed originally by Simon Tatham for the Windows platform. PuTTY is open source software that is available with source code and is developed and supported by a group of volunteers.

PuTTY.orgwww.putty.org

위 문장을 살펴보시면 PuTTY는 SSH와 telnet을 이용하는데 쓰이는 클라이언트이며, Simon Tatham이란 개발자가 만든 윈도우 전용 오픈소스 소프트웨어 라고 설명 되있네요.

윈도우에서 SSH를 사용하는데 가장 편한 툴이라고 생각합니다.

PuTTY 다운로드

아래 링크에서 최신버전의 PuTTY를 받을수 있습니다. 링크.

Xming

Xming is the leading X Window System Server for Microsoft Windows 8/7/Vista/XP (+ server 2012/2008/2003). It is fully featured, small and fast, simple to install and because it is standalone native Microsoft Windows, easily made portable (not needing a machine-specific installation).

Xmingsourceforge.net/projects/xming

홈페이지에는 위처럼 설명이 되어 있는데요, 간단히 PuTTY랑 같이 사용할수 있는 SSH용 GUI 입니다.
PuTTY와 함께 사용하면 SSH를 통해 오는 그래픽 출력을 가능케 해줍니다.

PuTTY에 Xming 연동하기

Run PuTTY, go under Connection->SSH->X11.
PuTTY에 연결(Connection)->SSH->X11 로 들어가 주세요


"Enable X11 forwarding" 이라고 되어 있는 항목을 체크 한다음, "X display location" 항목 안에 localhost:0 로 맞춰주시면 됩니다.
